# Contributing

感谢对 MacroCoach v2 的关注与贡献！

## 开发环境
- Python 3.10/3.11
- 安装依赖：`pip install -r requirements.txt`
- 运行应用：`streamlit run app.py`

## 代码规范
- 代码风格：`black .`
- 静态检查：`ruff check .`
- 提交信息：`type(scope): summary`（例：`feat(ui): add intake suggestion`）

## 测试
- 如果仓库包含测试：`pytest -q`

## 分支与 PR
- 分支命名：`feat/*`、`fix/*`、`docs/*`
- PR 请附：变更说明、动机/背景、截图（如涉及 UI）、影响范围、回滚方案
